# k6 performance testing POC

### [k6 instalation](https://k6.io/docs/getting-started/installation/)

### Setup

```bash
git clone https://github.com/danrusu/k6.git
cd k6
npm install
```

### Test execution

```bash
npm test
# OR
k6 run test/carsTest.js
```
